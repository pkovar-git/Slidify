Application Son's height prediction

This application provides prediction of son's height based on father's height. The prediction linear regression model was built on top of the father.son dataset provided within UsingR package.

Next slides show:

  1. Summary of father.son dataset

  2. Predictive lm model

    Up to 69.5inches(177cm) model predicts higher son..

  3. Quiz. The application gives you an answer.

Dataset father.son

Data set father.son contains 1078 pairs of heights (father, son) This is a summary.

Linear regression model

fit <- lm(sheight ~ fheight,data=father.son)

Intercept 33.89, slope 0.51
